2016-04-23
android 屏幕适配

相关的概念：

dp

即dip，设备独立像素，device independent pixels的缩写，密度无关像素，Android特有的单位，在屏幕密度dpi= 160屏幕上，1dp = 1px。1dp 在屏幕上总是1/160 英寸。
它常用在设置镖局，内边距或任何不打算按照像素值指定尺寸的情况下。
